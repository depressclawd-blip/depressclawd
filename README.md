# DepressClawd

Website dark & atmospheric (Vite + JavaScript vanilla), terinspirasi vibe situs seperti nwo.fun.

## Setup

```bash
npm install
npm run dev
```

Buka http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Stack

- **Vite** — build tool
- **Vanilla JS** — no framework
- **CSS** — custom (Syne + JetBrains Mono, green hacker accent, grid, noise)

## Struktur

- `index.html` — struktur halaman (header, hero, sections, footer)
- `src/main.js` — parallax mouse, scroll reveal
- `src/style.css` — variabel, layout, kartu, tombol
- `public/favicon.svg` — favicon

Edit teks, warna di `:root` di `style.css`, dan konten di `index.html` sesuai kebutuhan.
